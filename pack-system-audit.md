## System Check

1. Logo treatment: keep the large Sardibar wordmark centered on each front pack. The green, yellow, and orange variants should use the same scale, baseline, black outline weight, and top descriptor placement.
2. Illustration style: keep the hand-drawn sardine and flavor icons black-outlined and imperfect. Do not mix in polished vector fitness symbols or generic nutrition icons.
3. Accent colors: orange should own paprika/chili, yellow should own lemon/zaatar/sea salt, and green should own coriander/olive/sumac. Use the same red circular salt badge across all flavors.
4. Mascot usage: mascot and sardine illustration should feel like one family. Use the black-line sardine character on the site and supporting layouts; avoid replacing it with unrelated fish art.
5. Front-of-pack hierarchy: top claim, large logo, flavor line, sardine illustration, ingredient/cracker illustrations, nutrition side boxes, and weight should appear in the same order on all three packs.

## Precise Edits Still Recommended

1. Align the nutrition side boxes across all flavors so `12 גרם חלבון`, omega source, and calories sit at the same x/y positions and use the same border radius.
2. Normalize flavor copy length and placement. Orange currently reads denser than yellow and green; keep line breaks visually balanced.
3. Match the sardine drawing scale between the front-facing green file and the orange/yellow files.
4. Use one export style for final commerce cards: transparent or clean-cut pack render, consistent shadow, same canvas ratio, and no mixed background treatment.
5. Keep the close-up cracker assets as lifestyle/product detail images, not as the main front-of-pack ecommerce card image.
